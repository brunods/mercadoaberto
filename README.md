# Requisitos
- Docker
- PHP
- Composer
- Node
- NPM

# Iniciando o projeto
- `composer install`
- `npm install`
- `php artisan key:generate`
- Copie o env.example para .env
- `docker compose up -d`
- `npm run dev`
- `php artisan serve`
- `php artisan migrate`
- Acesse `localhost:8000/login` ou `localhost:8000/register`
